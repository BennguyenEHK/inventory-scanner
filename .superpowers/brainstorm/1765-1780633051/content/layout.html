A — Single-Page Pipeline

One long page: upload → stages animate downward as they complete → report appears at bottom → qty input → save. Everything visible in a single scrollable flow.

Pros

  • Zero navigation — scan to save in one place
  • Progress is always visible
  • Great for mobile

Cons

  • Page gets long with results
  • Hard to review past scans inline

B — Split Panel Dashboard

Left panel: upload + pipeline progress. Right panel: live results as they come in. Both visible at once. Nav tabs for Scan vs Inventory list.

Pros

  • See upload and results simultaneously
  • Professional dashboard feel
  • Efficient for power users

Cons

  • Cramped on mobile
  • Complex layout logic

C — Guided Wizard Steps

3 clear steps: Upload → Analysis runs → Review & save. Each step is a distinct card that expands as you complete the previous. Collapsed steps show status summary.

Pros

  • Crystal clear user flow
  • Each step focused, no overwhelm
  • Works well on all screen sizes

Cons

  • Less "live" feeling during pipeline
  • Extra click to see collapsed sections
